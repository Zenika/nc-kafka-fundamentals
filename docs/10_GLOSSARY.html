# Glossary

  • Apache Kafka (opens new window)

Apache Kafka is an open-source distributed event streaming platform used by thousands of companies for high-performance data pipelines, streaming analytics, data integration, and mission-critical applications.

  • Apache ZooKeeper (opens new window)

ZooKeeper is a centralized service for maintaining configuration information, naming, providing distributed synchronization, and providing group services.

  • Prometheus (opens new window)

Prometheus is an open-source systems monitoring and alerting toolkit

  • Grafana (opens new window)

Grafana allows you to query, visualize, alert on and understand your metrics no matter where they are stored. Create, explore, and share dashboards with your team and foster a data driven culture.

  • JMX (opens new window)

Java Management Extensions (JMX) is a Java technology that supplies tools for managing and monitoring applications

  • Apache Kafka JMX Metrics (opens new window)

  • Confluent &Control Center (opens new window)

Confluent Control Center is a web-based tool for managing and monitoring Apache Kafka
